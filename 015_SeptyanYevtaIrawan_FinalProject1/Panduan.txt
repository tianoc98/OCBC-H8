Untuk menggunakan Aplikasi : 
1. Buka XAMPP dan jalankan apache dan mysql
2. Jalankan dotnet run 
3. Akses local api menggunakan swagger UI : http://localhost:5000/swagger/index.html
4. Register dan login terlebih dahulu untuk mendapatkan token yang digunakan untuk Endpoint Payments dan Payments details

untuk menggunakan Setiap Endpoint

/api/AuthManagement/Register Method Post

-> Isikan 
 "username": "string",
 "email": "user@example.com",
 "password": "string" 

Password Harus menggunakan 1 Huruf besar , 1 Symbol , 1 angka minimal nya

-> Isikan 
 "email": "user@example.com",
 "password": "string"
